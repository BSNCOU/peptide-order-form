# Peptide Order Form

A web-based order form application for managing peptide product orders with automatic pricing calculations and order summary generation.

## Features

- **Interactive Order Form**: Add multiple product items to your order
- **Dynamic Pricing**: Automatic calculation of sales and public prices
- **Bulk Pricing**: Different pricing tiers for orders of 10+ bottles
- **Order Summary**: Generate and copy formatted order summaries
- **Responsive Design**: Mobile-friendly interface
- **Real-time Calculations**: Instant updates of totals and profit margins

## Usage

Simply open `index.html` in your web browser to use the application.

### How to Use

1. **Add Products**: Click "+ Add Product" to add items to your order
2. **Select Product**: Choose from the dropdown menu of available peptides
3. **Enter Quantity**: Specify the number of bottles
4. **View Pricing**: See instant sales and public pricing
5. **Generate Summary**: Click "Generate Order Summary" to create a formatted text summary
6. **Copy Order**: Use the "Copy to Clipboard" button to copy the summary

## Pricing

- Orders of **1-9 bottles**: Standard per-bottle pricing
- Orders of **10+ bottles**: Discounted bulk pricing (automatically applied)
